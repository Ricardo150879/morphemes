      • Lexical morphemes - Open Class of word
        • Nouns (girl, house)
        • Verbs (kick, look)
        • Adjectives (happy, long)
        • Adverb (quickly, evenly)
      • Functional morphemes - Closed Class of word
        • Derterminers
        • Conjunctions (and, but)
        • Prepostions (of, in)
        • Pronouns (I, he, you, they)
        • Auxiliary verbs (is)

    Noun Suffix Words List

    -ness -> state of being: heaviness, highness, sickness, dryness, coolness, baldness
    -acy -> quality: democracy, accuracy, lunacy, democracy, diplomacy, supremacy, autocracy, immediacy, obstinacy, intricacy, theocracy
    -ist -> person: geologist, protagonist, sexist, scientist, theorist, communist, receptionist, technologist, nutritionist, antifeminist, phlebotomist, editorialist,
    -al -> process of: remedial, denial, personal, trial, criminal, arrival, seasonal, global, magical, national
    -en -> become: sharpen, strengthen, loosen
    -tion -> state of being: position, promotion, ambition, pollution, revolution, affection, caution, option, solution, revolution
    -or -> person: creator, ablator, collaborator, inventor, abductor, irritator, franchisor
    -ty -> quality of: extremity, validity, enormity, susceptibility, territoriality, respectability, sentimentality, intentionality,
    -ment -> condition: enchantment, argument, adjustment, achievement, amazement, punishment, payment,
    -dom -> state of being: freedom, stardom, boredom, seldom, wisdom
    -ence -> quality of: ambiance, consequence, absence, convenience, experience, patience
    -ism -> belief: judaism, skepticism, escapism, exceptionalism, existentialism, despotism
    -ship -> position held: friendship, hardship, internship, flagship, hardship, township, lordship, kingship, starship, headship, wardship
    -er -> the object that does a specified action: teacher, interpreter, reader, writer, listener, reporter, watcher
    -ance -> state: tolerance, nuisance, importance, acceptance, assistance, appearance, compliance, resistance, attendance, reluctance

    Verb Suffix Words List

    -en -> become: sharpen, strengthen, loosen, brighten, darken, deepen
    -fy -> Make: Justify, simplify, magnify, satisfy, diversify, intensify, exemplify, indemnify, demystify, personify, speechify, electrify.
    -ate -> become: mediate, collaborate, create, accurate, appropriate, certificate, climate
    -ise -> Become: publicize, synthesize, hypnotize, advertise, advise, apprise, arise, chastise, circumcise, comprise, compromis

    -like -> similar to: lifelike, seamanlike, manlike, workmanlike, canelike, nutlike, sylphlike, apelike, trancelike.

    -ous -> full of: cerous, comous, cymous, famous, fucous, iodous, joyous, lupous, miaous, mucous, nodous, odious, osmous, parous, pilous

    Adverbs Suffix Words List

    -ly -> in a manner: quickly, badly, beautifully, boldly, bravely, breathlessly, carefully, certainly, correctly, dangerously, dutifully, eagerly, effortlessly, evenly, eventually
    -ward -> shows direction: forward, downward, backward, westward, eastward, windward, untoward, homeward, leftward.
    -wise -> in a manner of: crabwise, umbrellawise, castlewise, circularwise, flockwise, yokewise, zigzagwise, spirewise, choirwise, clockwise
    -ways -> shows direction: sideways, edgeways, fessways, lengthways, sunways, slantways
